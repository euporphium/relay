# Relay

Relay is a full-stack React application built on TanStack Start with a
type-safe database, authentication, and modern UI tooling.

## Features
- Full-stack application with integrated frontend and backend logic
- Type-safe data access and validation
- Built-in authentication and user session management
- Structured routing and server-side data loading for application workflows
- Shared UI component system for consistent, accessible design
- Developer-focused tooling for testing, linting, and rapid iteration
- Environment-based configuration and secrets handling

This repository is structured to support both human developers and
AI-assisted development through concise, domain-focused documentation.
